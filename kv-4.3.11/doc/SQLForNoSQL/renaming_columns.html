Renaming columns

To select lastname and rename it as Surname, use the AS keyword in the SELECT clause. 

sql-> SELECT lastname AS Surname FROM Users;
 +----------+
 | Surname  |
 +----------+
 | Scully   |
 | Smith    |
 | Morgan   |
 | Anderson |
 | Morrison |
 +----------+

5 rows returned
